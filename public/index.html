Well, if you are seeing this then, you probably have set this up correctly, but if needed here are some helpful tips and imformation. Creating Users: To create users, navigate to private\config\users.xml, copy a pre-existing user and change to params. Erase Data: Navigate to public\ delete the data dir and all files in it, then create a new folder and rename it to data. Best Practices: In file explorer, navigate to scripts\run.bat create a shortcut and add the shortcut to your desktop.
